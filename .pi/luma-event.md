## Event Context / Background

**What is Pi?**  
Pi is a minimalist, extensible coding agent harness. It powers OpenClaw and a growing ecosystem of developer configurations. Key philosophy: Bash is "all you need" as the interface. It's designed to be lightweight and configurable vs. heavier agents like Claude Code.

**Why this event?**  
The event brings together the Pi creator (Mario Zechner), key community figures (Armin Ronacher, Ben Vinegar), and the Toronto developer community (organized by Sentry + New) for hands-on learning, an AMA, and coworking. The timing aligns with growing interest in agentic coding tools.

**Sentry connection:**  
This project (`pi-sentry-monitor`) is in Sergiy Dybskiy's repository, and the Luma event is hosted under the Sentry calendar — Sergiy appears to be a Sentry employee organizing this community event.
